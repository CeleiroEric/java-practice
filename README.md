# java-practice

Pequeñas prácticas de Java, con ejemplos y experimentos para aprender.

---

## Prácticas incluidas

### Practice 1 - List Filter
**Objetivo:** Aprender a filtrar listas de objetos en Java.  
**Qué se trabajó:**
- Uso de bucles y condicionales para extraer elementos que cumplen ciertas condiciones.

### Practice 2 - Streams & Lambda
**Objetivo:** Introducción a las expresiones lambda y Streams en Java.  
**Qué se trabajó:**
- Filtrado, mapeo y reducción de colecciones usando Streams y funciones lambda.

### Practice 3 - Validation Util
**Objetivo:** Crear utilidades para validar datos.  
**Qué se trabajó:**
- Métodos de validación para strings y números.  

### Practice 4 - SQL-like Operations
**Objetivo:** Simular operaciones SQL sobre colecciones de objetos en Java.  
**Qué se trabajó:**
- Filtrado, ordenación y búsqueda de datos en listas, imitando consultas SQL.

### Practice 5 - CRUD Memory
**Objetivo:** Practicar operaciones CRUD (Create, Read, Update, Delete) sobre una lista de objetos `Usuario` en memoria.  
**Qué se trabajó:**
- **CREATE:** Añadir usuarios a la lista.  
- **READ:** Mostrar la lista antes y después de cada operación.  
- **UPDATE:** Modificar atributos de un usuario existente.  
- **DELETE:** Eliminar un usuario de forma segura, comprobando su existencia.
